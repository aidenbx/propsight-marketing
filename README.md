# PropSight Marketing Site

Next.js marketing site for PropSight — deploys to propsight.com.au via Vercel.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

Push to `main` — Vercel auto-deploys to the `syd1` (Sydney) region.

## Environment

| Variable | Description |
|---|---|
| `NEXT_PUBLIC_APP_URL` | URL of the PropSight app (default: `https://app.propsight.com.au`) |
